# Trabalho Pratico: Laboratório de Linguagem C
Laboratório de linguagem C

## **Objetivos**
 Aplicar lógica, estrutura de dados básicas e boas práticas; produzir codigo comentado, relatório de contribuição e apresentação oral demonstrativa. 

## **O que deve ser entregue**: 

### 1. Codigo Fonte
* Código **comentado**  e **identado** na linguagem C
2. Relatório Tecnico  
* **c. Funcionalidades e Lógica**  explicação dos principais algoritimos utilizados. 
* **d. Desafios e Soluções:** Dificuldades encontradas e como foram superadas
* **e. Contribuições individuais**  tabela com nome, tarefas realizadas, desafios de cada membro. 
* **f. Conclusão e aprendizados.** considerações finais sobre o projeto. 

###3. Apresentação oral 
* **Estrutura:**  Minimo **5 slides** 
* **Conteúdo obrigatório:**  Capa, problema resolvido, Demonstração  do funcionamento (demo), explicação de um  trecho de código desafiador e conclusão. 

# *Critérios de avaliação 

### 1. Código 
* **Funcionalidade:** o sistema rode sem erros. 
* **Organização** Código limpo, legível e bem estruturado.
* **Conceitos da diciplina** uso correto das estruturas como 'struct', vetores, etc. 
* **Tratamento de erros:** Validação de entradas inválidas do usuário. 
* **BÔNUS:** uso de arquivos, ponteiros ou alocação dinâmica. 
* **Utilização de Git/GitHub** para versionamento do código e histórico de commits.
### 2. Relatório 
* **Estrutura completa** Presença de todos os tópicos conforme solicitado.
* **Clareza na explicação** Qualidade e precisão na explicação do código. 
* **Tabela de contribuições** Detalhamento do papel de cada integrante.
3. Apresentação: 
* **Demonstração** Validação do sistema funcionando. 
* **Domínio tecnico** explicação clara da lógica de um trecho específico

## Observações 

* **Cópia**  Trabalhos com código idêntico entre duplas diferentes serão zerados. 
* **Dóminio do código** Se um integrande não souber explicar o código, a nota individual será aajustada. 
* **Plágio de um código**  da internet sem devida adaptação e entendimento receberão nota zero.
* **Caso utilize IA**  deve colocar no relatório: qual usou, como e o que entendeu.
